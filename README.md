# WebScoreBoard
These code can boost field game by recording players' score and show leaderboard.

## Prerequisite
See package.json
* bootstrap 3.4
* fontawesome 5.7.1
* jquery 3.3.1
* Node.js v10.15.1
* npm v6.4.1
* express 4.16.4
* express-session 1.15.6
* ejs 2.6.1
* bodyParser 1.18.3
* mongoose 5.4.10
* passport 0.4.0
* passport-local 1.0.0
